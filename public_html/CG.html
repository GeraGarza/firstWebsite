Computer Graphics

Skinning Animation

I created a skinned animation of a toy character. I implement basic skinning on the CPU. Assigning that each vertex is influenced by 18 bones. Using the current “time” variable, cheb advances the animation so that you see a dancing skeleton. Vertices are not influenced by all the bones. Most vertices are affected by at most 5 bones, and the maximum number of influences is 15. therefore, I created another attribute that stores the actual number of influences for each vertex.

Hierarchical Transformation

I create a robot that you can be manipulate with the keyboard, where you can also traverse through the body parts using depth first search.

Free Look World

I created a world invaded by chickens and bunnies. The chicken and bunny objects hold a transformation matrix that can easily translate/rotate/scale/shear the object in the world. The camera being controlled by the mouse, where the direction is controlled by the basic awsd keys.

Material Binding

I mapped a 2D image to a spherical object to simulate an Earth-like shape. Then added rotation based on time, where I also moved the sun to show the difference in the lighting.

Keyframed Animation

I created a keyframed animation of a toy helicopter, where the user determins the amount of key frames, and their location, and the simulation interpolates between these keyframes at a constant speed using arc-length parameterization. The path can be seen by using a cubic spline curve to show smooth transitions.
